About Me

Perkenalkan nama saya Yabes Maychel. Saya lahir di Pekanbaru pada tanggal 20 Mei 2002 tepatnya hari Senin. Saya biasanya dipanggil Yabes dan lebih singkat lagi bisa dipanggil Bes. Saya tinggal di Jalan Bambu Kuning 1 namun tidak ada nomor rumahnya karena belum mendapatkan data nya. Saya tidak memiliki banyak hobi karena saya lebih merasa nyaman di rumah saja. Hobi saya hanyalah bermain game dan futsal. Game yang saat ini paling saya sering mainkan adalah Valorant dan Mobile Legends. Saat ini saya sedang menempuh pendidikan di salah satu universitas di Indonesia yakni Universitas Riau. Saya mengambil program studi Teknik Informatika
